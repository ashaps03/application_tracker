# application_tracker
A smart-ish job application tracker that auto-fills applications using Gmail parsing and lets you manage them through a dashboard.

---

## 🛠️ Tech Stack
- **Frontend:** React (Vite)
- **Backend:** Flask (Python)
- **Database:** SQLite/Firebase
- **API Integration:** Gmail API
